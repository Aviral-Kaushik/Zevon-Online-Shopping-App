# Zevon-Online-Shopping-App
Zevon is an E-commerce platform for shopping. This Application includes all the features that are needed for an E-commerce application.

In this application, I have used Firebase real-time database for handling the user data and used Firebase cloud storage for storing images and videos of the user.

I have used Kotlin as a backend language for developing this application and used XML for making the UI of this application.

I have also implemented animation in this application for a better user experience.

In this application, a user can apply for becoming a seller and after becoming a seller the user can sell their product on Zevon.

In this application users can rate and write their views regarding a product concern.
